We are an on-demand software engineering team, accessible to clients located anywhere across the world.

This is possible because we work 99% remotely, usually from the US west coast time zone. For the remaining figurative 1% of our time, we travel anywhere around the world (client expense), for short missions (days) or in-person meetings.

We do software development, consulting in software engineering and applied Computer Science, and occasionally computer operation (sysadmin, dba, devops). We create software solutions favoring free(dom) and open source software (FOSS).

Our clients are sophisticated computer users able to communicate in writing effectively, with sporadic audio/video conversations when helpful. Writing enables asynchronous work rhythms which is both convenient and powerful with multi-timezone project participants. Note that we can communicate in French too.
