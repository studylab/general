Our software solutions rely extensively on free (as in freedom) and open source software (FOSS). Consequently our clients have access to the entire code base of their solutions with the valuable benefit of not being trapped in any closed computer code i.e. proprietary software.

As an integral part of our solutions is an effective technical documentation. When our clients need the services of other software shops, the open source software combined with an effective technical documentation makes it as smooth as a technology transfer can be.

The typical assignments we take on are:

  • code with effective documentation for programmers
  • proof of concept software
  • software architecture
  • maintenance contract for any code we produced
  • harnessing and combining open source tools
  • back-end software operation (sysadmin, dba, devOps)
  • evaluate the current software tool scenery that addresses an identified need
  • assess the feasibility of a computer solution
  • training and presentations on technical subjects or software tools
